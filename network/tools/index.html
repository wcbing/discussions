      IP 相关

      通用：

      • ip138
        国内很常用的一个
      • 纯真
        比较全的一个 IP 数据库

      本机：

      • ip111
        国内外双重查询（本机 IP）
      • 126 getLocByIp
        网易（126）的API，可以获取 IP 及地理位置
      • Sukka
        查询本机 IP 地址及归属地 | 查看 DNS 出口 IP
      • Sukka - 简版
        查询本机 IP 地址及归属地
      • IP查询
        IPV6 相关

      WHOIS 查询

      在线查询域名的相关信息，防止手头没电脑。

      • https://whoisw.com

      不会有人查不了个人 .cn 域名吧。 虽然 .top 也是中国某家企业运营的，但是总比不给隐私防护的 .cn 强吧。😁

      在线 Ping

      • ITDOG
      • 站长工具
        看各地的访问情况（尤其是国外网站有没有被q） 还能看域名是否有多个 IP

      DNS查询（域名查ip）

      • nslookup、dig、ping
      • https://www.itdog.cn/dns/
      • 等等，这种随便找
