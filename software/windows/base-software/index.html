      Windows装机必备

      Windows装机必备

      （2021.8）本文写给普通人，都是一些经常会用到而且容易有广告的。要求是开箱即用，操作简单。需要折腾的（或者找破解版）就不推荐了。

      软件安装

      Windows上下载软件一般推荐官网，但寻找和更新比较麻烦，而且下载速度差异很大。应用商店有基本的审核，比直接搜索要安全一点，就是有的不全、更新不及时。

      微软商店里面还是有不少东西的，但是国内常用的软件很多都没有。

      联想应用商店🔺 推荐
      腾讯软件中心 需要普通下载
      360软件宝库 需要普通下载

      如果是联想电脑，自带，但是我建议在设置中关闭【快装功能】，否则别问别人你软件装哪了。

      软件卸载

      列出所有“安装”的软件并启动它们的卸载程序，卸载完还会扫描有没有残留的文件夹、注册表等垃圾。

      这个是单文件，建议放到桌面，点开就能用。

      HiBit Uninstaller 建议下载Portable version（便携版）
      GeekUninstaller 很多人推荐，但功能有些简陋

      浏览器类（任选一两个即可）

      Firefox 火狐国际版，开源、自由、跨平台。我最常用的浏览器。
      注意最好下载国际版，具体看Firefox下载。
      Edge Windows自带。现在也算跨平台了。
      Chrome 国内环境用着很难受，难同步、默认Google（很多人真的不会改）。有些网站为了兼容性推荐你用Chrome，直接上面的Edge就行。非要用的话记得改搜素引擎（推荐Bing，广告少而且新建标签页漂亮）。
      360极速浏览器 唯一推荐的国产（大厂）浏览器。内核较新，广告较少。同步功能完善。
      有Linux版，虽然叫“360安全浏览器”但其实就是“极速”，密码同步方式不同。
      QQ浏览器 虽然不推荐但不少同学用，当然它的外观和翻译功能做得确实不错

      输入法

      Win10以上最好用自带的微软拼音。

      文档阅读

      MS office 几乎所有一线笔记本（联想、惠普、戴尔）和部分二线笔记本都带家庭版授权，记得及时激活，毕竟花了钱的。
      金山WPS 跨平台的国产办公软件，建议在不影响正常使用的情况下尽量“支持国产”。 如果嫌弃广告可以试试这个计算机等级考试官网的教育考试专用版。
      稻壳阅读器🔺 支持PDF、EPUB、MOBI、TXT等常见的电子书籍文档，还支持知网的.caj文件。
      甚至可以查看.md和.chm等，但效果不佳。

      本地看图/视频/听歌类

      自带播放器🔺 又不是不能用，Win10（较新版本）、Win11上自带的还不错。
      万能联播🔺 爱奇艺出品。视频、音频、图片都支持，基本没有广告。
      VLC 一个开源、自由、跨平台的播放器。去微软商店下载UWP版吧，原版太丑。

      不要推荐potplayer，要求开箱即用，我知道他能力很强，但他不符合国内大多数人的使用习惯。 它也有广告了，棒子的软件火了都会加广告

      压缩解压类（任选一个）

      360压缩 - 国际版
      7-Zip 或者在 联想软件商店 里下载。设置里有关联文件类型

      不推荐用BandiZip，韩国人开发的。基本就是个缝合怪，日常使用没有突出的优点，而且它现在也有广告。 WinRAR - 烈火修改版|App热 不太推荐修改版软件，但是某些人就爱这口，留个链接备用

      文件搜索

      Everything 一款文件、文件夹名称快速搜索软件

      安全管理类（任选一个）

      联想电脑管家 和火绒有合作，添加了软件管理和驱动管理等，没有品牌限制。
      火绒安全 用它主要是拦截弹窗和流氓软件的安装。推荐搭配上面的联想软件商店使用。
      360安全卫士（极速版） 结果还是有一个“360安全”。极速版无广告，老流氓还是能镇住小流氓的。

      远程控制

      向日葵远程控制 出问题解决不了了就让别人帮你解决

      再次提示

      注意不要直接百度然后用下载站的下载器，就是 “高速下载” 那个下载器，以免被捆绑安装。

      最好在软件商店安装，推荐联想软件商店（网页版也行）和微软商店。

      今年（2022）315曝光了，很多下载站都下架了“高速下载器”，就是不知道能维持多久
